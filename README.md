# 🎮 GameVerse

GameVerse is a browser-based gaming platform that offers a collection of fun and interactive games in one place. The project provides a smooth gaming experience with user authentication, profile management, leaderboards, and an attractive user interface.

### Features

- User Registration and Login
- User Profile
- Leaderboard
- Responsive Design
- Modern Gaming Interface

### Games

- Snake Game
- Tic Tac Toe
- Space Shooter
- Racing Car
- Hit The Mole
- Water Sort Puzzle
- X-O Game

### Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap
- LocalStorage

### How to Run

1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser or run it using Live Server.
